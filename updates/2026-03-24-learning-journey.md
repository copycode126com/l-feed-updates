我们把散落在日子里的努力，一点点捡回来，存好。这次更新带来全新的「学习旅程」页面，让每一天的付出都值得被记住。

### 主要功能

**学习旅程页面**
- **快速入口**：一键直达收藏单词、打卡历史、收藏文章、阅读历史——那些散落在各处的内容，现在集中展示
- **累计阅读词数**：统计你读过的每一个词，配合等级系统（Lv.0 到 Lv.5）和进度条，让积累可视化
- **年度热力图**：类似 GitHub 贡献图，记录过去一整年的打卡情况，绿色深浅代表学习时长
- **阅读趋势图**：累计曲线展示学习节奏，斜率变化帮你感知"这段时间我是怎么学的"

**账号管理功能**
- 新增账号注销功能，支持30天等待期内撤销
- 注销后所有数据清空，保障用户数据自主权

### 设计理念

这个功能不是用来"考核"你的，它更像一本日记。

- 热力图上某个周二下班很晚，但你还是读了15分钟——它记着
- 某个周末出去玩，回来还是打了卡——它也记着
- 也有一些空白的日子——都没关系

学英语像在黑暗里走路，你看不到终点，有时不知道是否在往前走。希望学习旅程能像路边的一盏小灯——在觉得"好像要放弃"的时候，回头看一眼，原来已经走了这么远。

### 其他优化

- **多语言支持**：中英文翻译完善，学习旅程页面全面国际化
- **文章页面优化**：阅读体验细节改进
- **关于页面更新**：新增产品介绍内容
- **个人信息页面重构**：样式与交互全面升级
